# COVID-19 Detection using Audio Data

This project focuses on detecting COVID-19 from audio signals using deep learning models.

## Features
- Audio preprocessing and feature extraction
- CNN + LSTM model
- Training and prediction pipeline

## Tech Stack
- Python
- Pandas, NumPy
- TensorFlow / Keras

## Files
- training.py → training model
- Predict.py → prediction
- preprocess.py → preprocessing
